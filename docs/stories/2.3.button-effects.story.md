## Story
**As a** usuário,
**I want** ver botões com design glass e efeitos visuais ao interagir,
**so that** cada clique seja visualmente satisfatório.

## Acceptance Criteria
1. Botões possuem: `background: rgba(255,255,255,0.05)`, `border: 1px solid rgba(255,255,255,0.1)`, `border-radius: 12px`, `color: #fff`
2. Botão `=` possui gradient vermelho destaque: `background: linear-gradient(135deg, #FF0000, #8B0000)`
3. Botões de operador (+, −, ×, ÷) possuem tom de vermelho sutil diferenciado dos numéricos
4. `:hover` aplica: `background: rgba(255,0,0,0.2)`, `box-shadow: 0 0 15px rgba(255,0,0,0.3)`, `transform: scale(1.05)`
5. `:active` aplica: `transform: scale(0.95)`, `box-shadow: 0 0 25px rgba(255,0,0,0.5)`
6. Todas as transições usam `transition: all 200ms ease`
7. Botão AC possui cor diferenciada (vermelho mais claro/brilhante)

## Dev Notes

**Prerequisite:** Stories 2.1 e 2.2 concluídas — Custom Properties e layout glass existem.

**Hierarquia de seletores CSS para botões:**
```css
/* 1. Base — todos os botões */
button { ... }

/* 2. Operadores — sobrepõe base */
button[data-action="operator"] { ... }

/* 3. Especiais — sobrepõem operadores */
button[data-action="equals"] { ... }
button[data-action="clear"] { ... }

/* 4. Pseudo-classes — estados interativos */
button:hover { ... }
button:active { ... }
button[data-action="equals"]:hover { ... }
```

**IMPORTANTE — `transform: scale()` e layout:**
- `scale(1.05)` no hover pode fazer botões "vazar" se não houver espaço
- Garantir que o grid tem `gap` suficiente para absorver a escala
- Alternativa mais segura: usar `box-shadow` crescente em vez de scale se houver problema

**Referência PRD:** FR10, FR11, UI Goals (Branding, Key Interaction Paradigms) — `docs/prd.md`

### Testing
- Hover em cada tipo de botão: numérico, operador, =, AC, +/−, %, ⌫
- Verificar glow vermelho visível no hover
- Verificar "encolhimento" satisfatório no click (scale 0.95)
- Testar no touch (mobile): `:active` deve funcionar em toque
- Verificar que `transition: 200ms` é perceptível mas não lento
